# Muhammad-Saquib-Portfolio 🎨✨

Welcome to the open-source repository for my personal portfolio! This is a highly interactive, modern, and design-driven web portfolio built to showcase my work in Graphic Design, Branding, Packaging, and Social Media Creative.

Designed with a sleek "Vismora" dark theme, editorial layouts, and fluid physics-based animations to create a stunning user experience.

## ✨ Key Features

- **Editorial "Magazine" Layouts**: A custom-designed *About* page featuring architectural grid lines, massive display typography, and asymmetrical content blocks.
- **Interactive UI & Physics**: Buttery smooth, physics-based animations powered by Framer Motion. 
- **Responsive Testimonials**:
  - **Desktop**: An infinite, smooth-scrolling marquee slider.
  - **Mobile**: A 3D stacked deck of interactive cards that users can physically swipe/drag to swap!
- **Dynamic Firebase Reviews**: A fully integrated backend allowing clients to leave reviews directly on the site, which are automatically fetched and injected into the testimonial slider.
- **Performance Optimized**: Built on Next.js App Router with server/client component separation for lightning-fast load times.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Library**: [React](https://react.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Database (Reviews)**: [Firebase Firestore](https://firebase.google.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

If you want to clone this repository and run it locally, follow these steps:

### 1. Clone the repository
```bash
git clone https://github.com/yourusername/Muhammad-Saquib-Portfolio.git
cd Muhammad-Saquib-Portfolio
```

### 2. Install dependencies
```bash
npm install
```

### 3. Setup Firebase (Optional for dynamic reviews)
If you want the dynamic review system to work, create a Firebase project, enable Firestore, and add your Firebase config to `src/lib/firebase.ts`.

### 4. Run the development server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
